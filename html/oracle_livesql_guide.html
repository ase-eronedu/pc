Oracle Live SQL (FreeSQL)로 시작하기

설치 없이 브라우저만으로 — 그림을 따라 클릭하면 끝나는 비전공자용 가이드

이라온에듀

Oracle Live SQL(요즘은 화면에 FreeSQL 이라는 이름으로 표시됩니다)은 오라클이 무료로 제공하는 브라우저 기반 실습 환경입니다. 내 컴퓨터에 아무것도 설치하지 않고, 실제 오라클 데이터베이스 위에서 SQL을 실행해 볼 수 있어 비전공자에게 가장 먼저 추천하는 방법입니다.

로그인 없이도 왼쪽에 미리 들어 있는 HR 샘플 데이터(직원·부서 등)를 둘러볼 수 있지만, 내가 직접 만든 테이블을 저장하려면 무료 오라클 계정으로 로그인해야 합니다. 아래 순서대로만 따라오세요.

이 가이드 흐름 (총 7단계)

접속 → 로그인/계정 만들기(②~⑤) → 내 테이블 만들기 → 조회

이미 오라클 계정이 있다면 STEP 2에서 로그인만 하고 STEP 6으로 건너뛰면 됩니다.

STEP 1

접속하고 화면 둘러보기

브라우저 주소창에 livesql.oracle.com 을 입력해 접속하면 아래와 같은 FreeSQL 화면이 열립니다. 왼쪽에는 연습용 HR 샘플 테이블이 보이고, 가운데는 SQL을 입력하는 워크시트입니다.

①

내가 만든 테이블을 저장하려면 화면 오른쪽 위 [Sign In] 을 클릭합니다.

STEP 2

로그인 또는 계정 만들기 시작

로그인 화면이 나옵니다. 이미 오라클 계정이 있다면 위 칸에 이메일을 입력하고 [다음]을 눌러 로그인한 뒤 STEP 6으로 넘어가세요.

①

계정이 없다면 아래 [계정 생성] 버튼을 클릭합니다. (이메일만 있으면 무료로 만들 수 있습니다.)

STEP 3

오라클 계정 만들기

계정 만들기 양식이 나옵니다. 별표(*)가 붙은 칸은 모두 필수 입력입니다.

①

이메일·암호·국가·이름 등 * 표시 항목 을 채웁니다. 회사명·주소·전화 등도 필수라 개인 학습용이라면 실제 정보로 적당히 채워 넣으면 됩니다.

②

맨 아래 동의 항목을 체크 한 뒤 [계정 만들기] 버튼을 클릭합니다.

Tip암호는 8자 이상이면서 대문자·소문자·숫자·특수문자를 모두 포함해야 합니다. (예: Sqld!2026)

STEP 4

이메일 인증하기

가입할 때 적은 메일함에 오라클이 보낸 인증 메일이 도착합니다.

①

메일 안의 [이메일 주소 확인] 버튼(또는 그 아래 URL)을 클릭합니다. 이 링크는 24시간 후 만료되니 받은 즉시 눌러 주세요.

STEP 5

계정 활성화 완료

“성공. 계정을 사용할 수 있습니다.” 화면이 보이면 계정 준비가 끝난 것입니다.

①

[계속] 버튼을 클릭합니다.

STEP 6

FreeSQL로 돌아와 내 테이블 만들기

다시 FreeSQL 워크시트로 돌아오면 로그인된 상태라 왼쪽 위가 [My Schema] 로 바뀝니다. 이제 내 전용 공간에 직접 테이블을 만들어 봅니다.

①

가운데 입력창에 아래 SQL을 그대로 붙여넣기 합니다. (회원 테이블을 만들고 데이터 2건을 넣는 코드)

CREATE TABLE MEMBER ( MEMBER_ID NUMBER, MEMBER_NAME VARCHAR2(30), GRADE VARCHAR2(10) ); INSERT INTO MEMBER VALUES (1, '홍길동', 'GOLD'); INSERT INTO MEMBER VALUES (2, '김철수', 'SILVER');
②

여러 줄을 한 번에 실행해야 하므로 [Run Script] 아이콘 (가로 줄이 그려진 두 번째 아이콘)을 클릭합니다. 키보드 F5 를 눌러도 똑같이 실행됩니다.

아래 Script output 에 Table MEMBER created 와 1 row inserted 가 나오면 성공입니다.

단축키 (실행을 더 빠르게)

F5 = Run Script — CREATE·DROP 같은 DDL이나 여러 문장을 한 번에 실행 Ctrl + Enter = Run — 커서가 놓인 한 문장(SELECT·INSERT 같은 DML)을 실행
STEP 7

내 테이블 조회하기

①

워크시트에 아래 조회(SELECT) 문 을 입력합니다.

SELECT * FROM MEMBER ORDER BY MEMBER_ID;
②

초록색 [Run] 버튼(▷) 을 클릭합니다. 키보드 Ctrl + Enter 를 눌러도 됩니다. (커서가 놓인 한 문장만 실행)

③

아래 Query result 에 홍길동·김철수 결과가 표로 나오면 완성입니다!

한 줄 정리

FreeSQL은 접속 → 로그인 → 내 테이블 만들기(Run Script) → 조회(Run), 이 흐름만 익히면 됩니다. 한 번 가입해 둔 오라클 계정은 다음 방법(SQL Developer 설치)에서도 그대로 사용합니다.
